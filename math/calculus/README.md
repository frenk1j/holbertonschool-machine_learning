## 📖 Description

This project covers the fundamentals of **Calculus** as applied to Machine Learning. It explores key mathematical concepts including summation and product notation, derivatives, partial derivatives, and both definite and indefinite integrals — building the mathematical foundation essential for understanding ML algorithms.

---

## 🎯 Learning Objectives

By the end of this project, you should be able to explain the following **without the help of Google**:

### Notation & Series
- ∑ Summation and ∏ Product notation
- What is a series?
- Common series and their properties

### Derivatives
- What is a derivative?
- The **Product Rule**
- The **Chain Rule**
- Common derivative rules
- What is a **partial derivative**?

### Integrals
- What is an **indefinite integral**?
- What is a **definite integral**?
- What is a **double integral**?
