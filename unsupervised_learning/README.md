# Unsupervised Learning

This directory contains projects and tasks related to unsupervised learning in machine learning. It focuses on methods used to identify patterns, structures, and relationships in data without relying on labeled outputs.

## Overview

The purpose of this folder is to organize unsupervised learning projects completed as part of the Holberton School Machine Learning curriculum. Each subdirectory represents a specific topic and includes the corresponding tasks, exercises, and implementations.
